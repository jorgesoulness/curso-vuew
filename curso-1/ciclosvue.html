Eso sirve para entrar a la instancia en los estados en lo que se tengan que ingresar mediante métodos.
Cuando se inicia la aplicación con new Vue se ejecutan los siguientes métodos:

  • Metodo: beforeCreate() Este es el prime método que se inicia automaticamente justo antes de crear la instancia
  • Metodo: created() Despues de que se crea la instancia, se ejecuta created.
  • Metodo: beforeMount() Este medoto se ejecuta justo antes de que se monte la pantilla en el DOM de la aplicación
  • Metodo: mounted() Se ejecuta cuando se haya montado la plantilla en el DOM
  • Metodo: beforeUpdate() Se ejecuta antes de que actualice algo, como alguna propiedad, metodo, función, etc.
  • Metodo: update() Se ejecuta justo cuando se haya hecho la actuaización
  • Metodo: beforeDestroy() Se ejecuta antes de destruir la instancia
  • Metodo: destroyed() Se ejecuta cuando la instancia se destruye
